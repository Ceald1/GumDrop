# GumDrop
Linux rootkit research project (kinda started out as vibe-coded and as a C learning opportunity). 


## Building:
1. `make`
2. `sudo insmod gumdrop.ko`

## Removing:
`sudo rmmod gumdrop`

## Use:
* `kill -42 12345` or any PID will grant the process that ran kill root access.
